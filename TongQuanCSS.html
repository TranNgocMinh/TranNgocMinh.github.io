Tổng quan về CSS

Mục tiêu

Kiến thức:

  • Trình bày được khái niệm CSS;
  • Giải thích được tại sao phải sử dụng CSS trong thiết kế trang web;
  • Trình bày được các hình thức kết hợp CSS vào trang HTML.

Kỹ năng:

Thao tác được các hình thức kết hợp CSS vào trang HTML.

Năng lực tự chủ và trách nhiệm:

Có ý thức tự giác, tính kỷ luật cao, tinh thần trách nhiệm trong học tập.

Yêu cầu

  • Đảm bảo kiến thức và kỹ năng từ Bài mở đầu và Bài 1
  • Có (hoặc đăng ký mới) tài khoản Github tại https://github.com/

Nội dung

1) Khái niệm chung

a) Khái niệm CSS

CSS viết tắt của từ Cascading Style Sheets là ngôn ngữ dùng để mô tả cách thể hiện của một trang web (theo https://www.w3.org/standards/webdesign/htmlcss.html)

b) Tại sao phải sử dụng CSS

CSS hỗ trợ định dạng nội dung (content), bố cục (layout), thiết kế (design) và cách thức trang web hiển thị trên các thiết bị có kích cỡ khác nhau (Responsive Web). Ngôn ngữ HTML hoặc hỗ trợ rất ít hoặc không hỗ trợ các vấn đề nêu trên.

Sử dụng CSS cho phép tách riêng phần nội dung (cái gì?) và hình thức (trông như thế nào?). Điều này giúp cho các nhà thiết kế web dễ dàng quản lý tài nguyên dự án và sẽ tốn ít thời gian khi viết hay chỉnh sửa mã trang web.

2) Các hình thức kết hợp CSS vào trang HTML

Cho trang index.html có nội dung như sau:

a) Sử dụng thuộc tính style (inline)

Định nghĩa CSS dùng thuộc tính style của một phần tử xác định.

Ví dụ: Định nghĩa CSS cho thẻ p

Giải thích:

font-size và color được gọi là các thuộc tính (attributes) CSS; 30px và blue là các giá trị tương ứng cho các thuộc tính này. Chi tiết về các thuộc tính CSS và giá trị của chúng sẽ được đề cập trong tiết học sau.

Kiểm tra thực thi

Thực hành số 1

  • Truy cập đến trang github cá nhân (ví dụ: github.com/ABC)
  • Mở (hay tạo mới) một repository tên ABC.github.io
  • Tạo tập tin inline.html với nội dung có thể sao chép tại đây.
  • Thêm thuộc tính style đến thẻ h1 kèm với định nghĩa CSS color:red;
  • Commit tập tin
  • Kiểm tra nhanh bằng cách truy cập ABC.github.io/inline.html

b) Sử dụng thẻ <style> (internal)

Định nghĩa CSS trong thẻ <style>. Thẻ <style> thường được đặt trong thẻ <head>.

Ví dụ: Định nghĩa CSS cho thẻ p

Giải thích:

Các thuộc tính CSS và giá trị tương tự trong Thực hành số 1 nhưng phần định nghĩa CSS được đặt trong cặp ngoặc {} theo sau thẻ p. Chi tiết về cú pháp trình bày CSS sẽ được đề cập trong các tiết học sau.
